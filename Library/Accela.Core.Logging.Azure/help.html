Logging Component Coniguration Items(Sample):

In App configuration(web.config/app.config), Add Below settings:

1. Log_DBConnectionString: Log Database connection string. When host web api to query log data, the host server needs to set this value.
Please replace {dbServer},{userid},{password} with correct values.


2. Log_StorageConnectionString:
Log is stored to storage temporarily, then a worker role will batch read the logs from storage and batch insert to database.
The log client consumer needs to set the value.
Please replace 'UseDevelopmentStorage=true' with correct value.
